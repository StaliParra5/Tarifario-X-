# X-API n8n Documentation

Este es un proyecto React que proporciona documentación interactiva sobre la integración de la API de X (anteriormente Twitter) con n8n, centrándose en el modelo Pay-Per-Use implementado en 2026.

## Características

*   **Interfaz moderna** con diseño oscuro y navegación intuitiva.
*   **Calculadora interactiva** diseñada para modelar escenarios de costos.
*   **Documentación estructurada** que detalla la API de X y su integración con n8n.
*   **Diseño responsive** para una visualización óptima en dispositivos de escritorio y móviles.
*   **Animaciones suaves** y transiciones elegantes para una experiencia de usuario fluida.
